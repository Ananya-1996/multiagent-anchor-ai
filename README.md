
# 🌿 Daily Support Assistant

An AI-powered **multi-agent system** designed to help users manage tasks, schedules, and emotional well-being — especially during challenging times like job loss.

---

## 🚀 Overview

The **Daily Support Assistant** combines emotional intelligence with practical task execution.  
It allows users to interact through a simple chat interface and get help with:

- 💙 Emotional support  
- 📅 Scheduling meetings  
- 📝 Creating notes  
- 🌿 Generating daily routines  
- 🌤 Planning based on weather  
- 💡 Smart suggestions  

---

## 🧠 Key Features

### 💙 Support Agent
Provides empathetic and responsible AI-based responses to help users cope with stress or anxiety.

### 🌿 Routine Agent
Generates structured daily routines with small, achievable steps.

### 📅 Calendar Agent
Schedules meetings and suggests available time slots with Google Meet links.

### 📝 Notes Agent
Creates and stores structured notes for better organization.

### 🌤 Weather Agent
Provides weather insights to help plan the day effectively.

### 💡 Suggestion Agent
Recommends next steps and helps maintain productivity.
